Quick Start
============

Installation
-------------

To install stable-worldmodel with minimal dependencies, run:

.. code-block:: bash

   pip install stable-worldmodel

To install stable-worldmodel with all optional dependencies, run:

.. code-block:: bash

   pip install "stable-worldmodel[all]"
